## How to run the App
  ### Install dependencies
  First the backend:
 ```
    1. cd backend

    2. npm install
    
    3. npm run dev
  ```

  Then the frontend:
  ```
  1. cd frontend

  2. npm install
  
  3. npm run dev
  ```
